# humanizeMs()

The `humanizeMs()` function converts a number of milliseconds into the largest applicable
unit of either year, month, week, day, minute, or second by returning a tuple of resulting
number with requested decimal places in first position and respective human-readable unit
in second position.

### Usage

```ts
import { humanizeMs } from '@spa-tools/utilities';

const humanizedMs1 = humanizeMs(1000);
console.log(humanizedMs1);
// --> ['1.0', 'second']

const humanizedMs2 = humanizeMs(1000 * 60 * 60);
console.log(humanizedMs2);
// --> ['1.0', 'hour']

const humanizedMs3 = humanizeMs(1000 * 60 * 60 * 24 * 365);
console.log(humanizedMs3);
// --> ['1.0', 'year']
```

### Arguments

| Name            | Type     | Required? | Default | Description                           |
| --------------- | -------- | --------- | ------- | ------------------------------------- |
| `milliseconds`  | `number` | yes       | -       | The milliseconds to humanize          |
| `decimalPlaces` | `number` | no        | `1`     | The decimal places to round/format to |

### Returns

Returns humanized milliseconds as `string`.
