<h3>Intent</h3>

The Core Router is intended for real-world applications that _neither require nor desire_ SEO
optimization. If your app depends heavily on SEO, then you should consider using a SSG/SSR
solution that takes care of routing+SEO (e.g. Next, Gatsby, Nuxt, Hugo, Docusaurus, etc.)
